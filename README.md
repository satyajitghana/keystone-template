# Keystone Template

App built with `yarn create keystone-app`

Test DB Connection with: `psql postgres://postgres:postgres@postgres_db/`

## Run

To launch your app, run:

- cd myapp6
- yarn install
- yarn dev

## To delete the volume

```text
docker compose down --volumes
```
